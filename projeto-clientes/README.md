# Projeto Clientes

Sistema simples de cadastro de clientes desenvolvido em Python, utilizando banco de dados SQLite.

## Funcionalidades
- Cadastrar clientes
- Listar clientes
- Atualizar clientes
- Excluir clientes
- Persistência de dados em banco de dados

## Tecnologias utilizadas
- Python 3
- SQLite

## Estrutura do projeto
projeto-clientes/
├── banco.db
├── conexao.py
├── crud.py
├── main.py
└── README.md


## Como executar o projeto
1. Clone o repositório
2. Acesse a pasta do projeto
